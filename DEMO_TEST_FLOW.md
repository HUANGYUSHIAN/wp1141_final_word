# Demo 測試流程（5分鐘）

## 📋 總覽
本測試流程涵蓋學生、廠商、管理員三個角色的核心功能，總時長約 5 分鐘。

---

## 🎓 第一部分：學生功能測試（約 2 分鐘）

### 1.1 登入與首頁瀏覽（15秒）
- [ ] 使用學生帳號登入
- [ ] 查看學生首頁，展示 6 個功能卡片：
  - 我的單字本
  - AI 生成單字本
  - 單字測驗
  - 單字遊戲
  - 單字複習
  - 點數兌換

### 1.2 單字本管理（30秒）
- [ ] 點擊「我的單字本」
- [ ] 點擊右上角「+」按鈕或「新增單字本」按鈕
- [ ] 建立新單字本：
  - 輸入單字本名稱（例如：「TOEIC 單字」）
  - 選擇使用語言（例如：英文）
  - 選擇解釋語言（例如：中文）
  - 選擇是否公開
  - 點擊「建立」
- [ ] 在單字本中新增幾個單字（例如：apple, banana, orange）
- [ ] 展示單字列表

### 1.3 AI 生成單字本（20秒）
- [ ] 返回首頁，點擊「AI 生成單字本」
- [ ] 輸入主題（例如：「餐廳用餐」）
- [ ] 選擇語言設定
- [ ] 點擊生成，展示 AI 生成的單字本

### 1.4 單字測驗（20秒）
- [ ] 返回首頁，點擊「單字測驗」
- [ ] 選擇一個單字本
- [ ] 進行一次測驗（回答 2-3 題即可）
- [ ] 展示測驗結果

### 1.5 點數兌換（15秒）
- [ ] 返回首頁，點擊「點數兌換」
- [ ] 查看可用優惠券列表
- [ ] 點擊一個優惠券查看詳情
- [ ] 展示 Google Maps 連結功能（點擊地址跳轉）

---

## 🏪 第二部分：廠商功能測試（約 1.5 分鐘）

### 2.1 登入與首頁瀏覽（10秒）
- [ ] 登出，使用廠商帳號登入
- [ ] 查看廠商首頁，展示 3 個功能卡片：
  - 優惠券管理
  - 店鋪資訊
  - 設定

### 2.2 店鋪資訊管理（30秒）
- [ ] 點擊「店鋪資訊」或「設定」
- [ ] 編輯預設店鋪資訊：
  - 輸入店名（例如：「五九麵館」）
  - 輸入地址（例如：「106臺北市大安區新生南路三段88之2號」）
  - 輸入營業時間
  - 點擊「在 Google Maps 中查看」測試地址跳轉
- [ ] 新增分店：
  - 點擊「新增分店」
  - 輸入分店名稱、地址、營業時間
  - 儲存
- [ ] 展示分店列表

### 2.3 優惠券管理（50秒）
- [ ] 點擊「優惠券管理」
- [ ] 新增優惠券：
  - 點擊「新增優惠券」
  - 輸入優惠券名稱（例如：「滿百折十」）
  - 選擇使用期限
  - **選擇分店**（展示分店選擇功能）
  - 輸入優惠內容
  - 儲存
- [ ] 查看優惠券列表，確認顯示分店名稱
- [ ] 點擊「編輯」修改優惠券
- [ ] 展示優惠券擁有者列表（如果有）

---

## 👨‍💼 第三部分：管理員功能測試（約 1.5 分鐘）

### 3.1 登入與首頁瀏覽（10秒）
- [ ] 登出，使用管理員帳號登入
- [ ] 自動跳轉到用戶管理頁面
- [ ] 查看左側導航欄：
  - 用戶管理
  - 單字本管理
  - 優惠券管理
  - 回饋管理
  - 系統設定

### 3.2 用戶管理（30秒）
- [ ] 在用戶管理頁面：
  - 查看用戶列表（展示分頁功能）
  - 點擊一個用戶查看詳情
  - 展示編輯用戶功能（修改名稱、角色等）
  - 展示用戶鎖定/解鎖功能

### 3.3 單字本管理（25秒）
- [ ] 點擊「單字本管理」
- [ ] 查看單字本列表
- [ ] 點擊一個單字本查看詳情
- [ ] 展示編輯/刪除功能
- [ ] 展示單字本上傳功能（Excel 匯入）

### 3.4 優惠券管理（20秒）
- [ ] 點擊「優惠券管理」
- [ ] 查看所有優惠券列表
- [ ] 點擊一個優惠券查看詳情
- [ ] 展示編輯/刪除功能

### 3.5 系統設定（5秒）
- [ ] 點擊「系統設定」
- [ ] 快速展示系統設定頁面

---

## 🎬 結尾（10秒）
- [ ] 返回首頁
- [ ] 展示 AI 助手功能（點擊右下角機器人圖示）
- [ ] 簡短展示 AI 助手對話功能

## 📝 測試注意事項

### 準備工作
1. **測試帳號準備**：
   - 學生帳號（已有一些單字本和點數）
   - 廠商帳號（已有一些優惠券和分店）
   - 管理員帳號

2. **測試資料準備**：
   - 準備幾個測試用的單字本
   - 準備幾個測試用的優惠券
   - 確保有測試用戶資料

3. **功能確認**：
   - 確認 Google Maps API 正常運作
   - 確認 AI 助手功能正常
   - 確認所有頁面載入正常

### 錄製建議
1. **畫面切換**：使用平滑的過渡，避免突然跳轉
2. **操作速度**：保持適中速度，確保觀眾能看清楚操作
3. **重點標示**：在重要功能上稍作停留，可以口頭說明
4. **錯誤處理**：如果遇到錯誤，可以快速說明並繼續

### 口述腳本建議
- **開場**：「今天要為大家展示一個全端背單字與抽獎券服務平台，這個平台提供學生、廠商、管理員三種角色的完整功能。」
- **學生部分**：「首先展示學生功能，學生可以建立單字本、使用 AI 生成單字、進行測驗，並用點數兌換優惠券。」
- **廠商部分**：「接下來展示廠商功能，廠商可以管理多個分店資訊，並發行優惠券給學生使用。」
- **管理員部分**：「最後展示管理員功能，管理員可以管理所有用戶、單字本、優惠券等系統資源。」
- **結尾**：「系統還提供 AI 助手功能，協助使用者快速上手。」
